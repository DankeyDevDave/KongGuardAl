Testing Instructions

Use these endpoints to test your Kong Guard AI plugin's threat detection capabilities:

  • Rate Limiting: Make rapid requests to /api/rapid
  • Suspicious Access: Try accessing /admin or /.env
  • Error Patterns: Generate errors using /api/error/* endpoints
  • Large Payloads: Send large POST requests to /api/upload
